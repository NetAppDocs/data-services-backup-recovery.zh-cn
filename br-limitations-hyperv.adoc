== 不支持的操作

NetApp Backup and Recovery中的 Hyper-V 工作负载的私人预览版不支持以下操作：

* 使用来自多个 Hyper-V 主机的虚拟机创建资源组
* 将虚拟机还原到备用位置
* 跨磁盘（跨多个 CIFS 共享）
* 通过 SAN 保护虚拟机
* 编辑保护组
* 保护使用系统中心虚拟机管理器 (SCVMM) 集群的 Hyper-V 主机
