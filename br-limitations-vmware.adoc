= NetApp Backup and Recovery针对 VMware 工作负载的已知限制
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
此处列出了与此版本不兼容或不能很好地兼容的平台、设备或功能。仔细阅读这些限制。

NetApp Backup and Recovery中的 VMware 工作负载预览版不支持以下操作：

* 挂载
* 卸载
* 附加 VMDK
* 分离 VMDK
* vVol 支持
* NVMe 支持
* 电子邮件集成
* 编辑政策
* 编辑保护组
* 基于角色的访问控制 (RBAC) 支持
