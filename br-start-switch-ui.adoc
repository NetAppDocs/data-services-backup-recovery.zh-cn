== 切换到不同的工作负载

您可以在NetApp Backup and RecoveryUI 中切换到不同的工作负载。

.步骤
. 从控制台左侧导航中，选择*保护*>*备份和恢复*。
. 从页面右上角，选择*切换工作负载*下拉列表。
. 选择您想要切换到的工作负载。
+
页面刷新并显示所选的工作负载。
