= NetApp Backup and Recovery针对 Oracle 工作负载的已知限制
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
此处列出了与此版本不兼容或不能很好地兼容的平台、设备或功能。仔细阅读这些限制。

NetApp Backup and Recovery中的 Oracle Database 工作负载的私人预览版不支持以下操作：

* 离线备份
* 克隆
* ASM 配置
* 保护存储在 SAN 上的数据库


Oracle 数据库仅支持在 Oracle 工作负载的私人预览版中使用 NFS 作为独立部署。
